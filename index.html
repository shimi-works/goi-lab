語彙ラボ

日本語の語彙力を、暇つぶしで育てる APIキー未設定 ？

ようこそ、語彙ラボへ 📖

日本語の語彙を「暇つぶし」で増やすツールです。使い方はかんたん：

  1. おまかせで難しめの語を10個出し、知らない語を語彙帳へ登録（レベル・ジャンルも選べます）。
  2. 調べるで気になった語を検索。意味・類義語・対義語・例文・使われる場面まで出ます。
  3. 語には認知度（知らない／あやふや／知ってる／覚えた！）を付けて管理。
  4. クイズや今日の10問で復習。正解すると「覚えた！」に昇格し、忘却曲線で最適な間隔でまた出題されます。
  5. 語彙帳で成長グラフ・🔥連続日数・ジャンル別に、伸びを確認。

右上の「APIキー」からGeminiの無料キーを入れると全機能が使えます。お試し用のサンプル語を数個入れてあります。ヘッダーの ？ でいつでもこの説明を開けます。

📖 詳しい使い方
思いつかない時は→ おまかせタブで難語を10個出す
レベルを選んでボタンを押すと、その難易度の語を10個出します。
ChatGPT等で調べた単語を貼り付け→まとめて登録。1行1語でも「単語：意味」でも、番号・カンマ区切りでもOK。
（コツ：ChatGPTに「覚えたい単語をリストにして」と頼むと貼りやすい）
「蝶と蛾の違い」のような雑学・違い・まとめは、単語ではなく豆知識としてここに。ChatGPTの答えを貼り付けてもOK。
データはこの端末に保存。念のため定期的に→

Gemini APIキーを設定（無料）

このアプリは Google の Gemini（無料枠・クレカ不要） を直接呼び出します。キーはこの端末の localStorage にのみ保存され、外部には送りません（Googleへの通信を除く）。共有PCでは注意。
キー取得: aistudio.google.com/apikey（Googleでログイン→「Create API key」）
