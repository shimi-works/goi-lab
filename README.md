# 語彙ラボ / Goi Lab

日本語の語彙力を「暇つぶし」で育てる、単一HTMLの学習ツール。
調べた語が自分の語彙帳に貯まり、忘却曲線に沿って復習が回りだす。

- アプリを開く … `index.html`（GitHub Pages: /goi-lab/）
- 技術説明・設計思想 … `portfolio.html`（/goi-lab/portfolio.html）

## 特徴
- 単一HTML・ノーサーバー・localStorage 永続化
- Gemini API をブラウザから直接呼び出し（各自の鍵を入力＝BYOK）
- 認知度4段階／忘却曲線／クイズ／今日の10問／成長グラフ
- 無料枠対策（上限で自動モデル切替・結果キャッシュ・自動リトライ）
